# TODO (Task) API
## Requirement
- MySQL
- Node.js

## Install 
pull repository

```
git pull https://github.com/besthalo/todo.git
```

go to project Directory
```curl
cd ./{project_name}
```

install package , use npm or yarn 

```curl
npm install
```

start service

```curl
node server.js
```


test service:
[Health Check](http://localhost:3000/v1/healthcheck)
